AdAuctionを使う準備（2013/1/1更新）

1.ads.csvを更新
追加する広告のid, budget, cpaを入力（このbudgetは使わないけど笑）

2.icvr.csvを更新
広告，広告枠，icvrを入力
icvrのデータがない場合は1e-09で補完
意図的にめちゃくちゃ低いicvrにすることで，入札させない
結果、閲覧者情報から入札すべきでない枠に対し入札することを防ぐことができる
（松原先生に微妙って言われたorz、icvr小さすぎるとエラー起きるのでダメ）
ないやつは考えなくていいとのことだったので，無視する

3.start.csvを更新
広告，広告枠，インプレッション数，総予算，実際の入札回数を入力
広告Aが広告枠Bで獲得したインプレッション数，それに支払った落札額の合計という意味
現在は2012/6/29のインプレッションデータを使用

4.adspacesディレクトリに広告枠の入札額-インプレッション数関数のcsvファイルを設置
これはサーバーから頑張ってデータを引っ張ってくる（今ここでめちゃくちゃ時間使ってる）

5.Organizer.javaのadsとadSpacesをいじる
重複しない，広告，広告枠ともに1つ以上であれば正常に動作するはず
